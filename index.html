django-jqueryfileupload

Django app for handle multiple file uploads via jquery-file-upload plugin.

View On GitHub
Downloads:
  • ZIP
  • TAR

    Installation

    pip install django-jqueryfileupload

    Configuration

    Optional

    # settings.py
INSTALLED_APPS += ('jqueryfileupload',)

    Add Blueimp Jquery File Upload plugin to static path.

    Usage

    # urls.py
from jqueryfileupload.views import UploadFileView, DeleteFileView
from myapp.models import MyModel

urlpatterns += (r'^imageuploader/upload/$',
                UploadFileView.as_view(model=MyModel,
                                       file_field='image'))

urlpatterns += (r'^imageuploader/delete/(?P<pk>\d+)/$',
                DeleteFileView.as_view(model=MyModel))
